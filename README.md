# World Checkers

**World Checkers** — современная веб-платформа для игры в шашки, созданная как полноценный продукт, а не просто статическая доска. Проект ориентирован на игроков, которые хотят быстро начать партию, тренироваться, решать задачи, изучать стратегии и соревноваться с другими пользователями. 

## Проблема на которй базируется идея
На рынке цифровых платформ для настольных интеллектуальных игр наиболее развитым и коммерчески успешным направлением остаются шахматы. В этой сфере уже существуют сильные продукты с высоким уровнем пользовательского опыта, развитой экосистемой и широкой аудиторией: например, Chess.com, World Chess и Arena Chess, последняя из которых имеет официальную сертификацию FIDE.

В сегменте шашек ситуация заметно отличается. На данный момент одним из немногих заметных онлайн-проектов является Lidraughts, однако платформа имеет сравнительно низкий онлайн, ограниченную узнаваемость и не обладает прямой официальной поддержкой со стороны FMJD — международной федерации шашек, выполняющей для шашек роль, аналогичную FIDE в шахматах.

Идея нашей платформы заключается в создании современной, удобной и масштабируемой экосистемы для игры в шашки, ориентированной как на начинающих пользователей, так и на профессиональных игроков. В качестве референсов были взяты такие проекты, как Chess.com и World Chess, где сочетаются качественный дизайн, удобный интерфейс, соревновательная среда и дополнительные образовательные функции.

Ключевое отличие нашего продукта — концепция platform by FMJD: онлайн-платформы, которая может быть официально связана с международной федерацией шашек и использоваться не только для игры, но и для проведения рейтинговых турниров, подтверждения результатов и получения официальных онлайн-титулов. Подобная модель уже реализована в шахматной индустрии, например в World Chess, и может стать важным шагом для популяризации шашек на глобальном уровне.

Таким образом, проект направлен не просто на создание очередного сайта для игры в шашки, а на формирование полноценной цифровой инфраструктуры, которая сделает шашки более доступными, современными и привлекательными для международной аудитории.

## Идея продукта

Большинство онлайн-шашек ограничиваются простой доской и базовой логикой ходов. **World Checkers** делает акцент на полноценном пользовательском опыте:

- игра по разным правилам шашек, в том числе международных шашек 10x10 и русских шашек 8x8;
- современный интерфейс в стиле онлайн-шахматных платформ;
- обучение, задачи и тренировка мышления;
- рейтинги, турниры и прогресс игрока;
- возможность расширения до мультиплеера, AI Coach и монетизации.

Главная цель проекта — показать, что шашки могут быть не только настольной игрой, но и полноценной онлайн-платформой для обучения, соревнований и развития стратегического мышления.

## Основной функционал

### Игровой процесс

- Интерактивная доска для шашек.
- Поддержка международных шашек 10×10.
- Логика передвижения фигур.
- Взятие фигур соперника.
- Превращение шашки в дамку.
- Определение состояния партии.
- Компоненты для игрового поля 8×8 и 10×10.

### Пользовательский опыт

- Главная страница с презентацией продукта.
- Страница игры.
- Страница выбора режима игры.
- Профиль пользователя.
- Адаптивный дизайн для компьютеров, планшетов и телефонов.
- Переключение светлой и тёмной темы.

### Обучение и удержание пользователей

- Раздел с уроками.
- Раздел с задачами / puzzles.
- Возможность добавления подсказок и анализа ходов.
- Основа для истории партий и прогресса игрока.

### Социальные и соревновательные функции

- Лидерборд игроков.
- Турнирная страница.
- Основа для рейтинговой системы ELO.
- Возможность развития до глобальных рейтингов по городам и странам.

### Backend и аккаунты

- Подготовлена интеграция с Supabase.
- Подготовлены страницы регистрации и входа.
- Возможность хранения пользователей, рейтингов, партий и прогресса.

## Стек технологий

- **Vue.js 3** — frontend framework.
- **Vite** — быстрый dev server и сборка проекта.
- **JavaScript** — основная логика приложения.
- **CSS** — стилизация интерфейса.
- **Supabase** — backend, база данных и авторизация.
- **Git / GitHub** — контроль версий и публикация кода.

## Почему это ценно

**World Checkers** может быть полезен:

- игрокам, которые хотят получить официальный титул world checkers;
- игрокам, которые хотят тренироваться и решать задачи;
- детям и подросткам для развития стратегического мышления;
- организаторам турниров;
- пользователям, которым нужен быстрый онлайн-матч с другом.

Проект можно развивать как образовательную и соревновательную платформу, а не просто как одноразовую игру.
